\begin{frame}{Samples}

  \item Alternate sample (4FS, massive b): Zbb MiNNLO.
  \item We now use the correct event weights from the LHE files: thanks to Marius, Kenneth for setting this up.
  \item Marco Zaro confirmed that in arXiv1803.04336 they used B hadrons to swap, as we do here.
\end{itemize}
\end{frame}

\begin{frame}{Event Selection and Swap Definition}
\begin{itemize}
  \item Event selection:
  \begin{itemize}
    \item prefsrLep\_pt $>$ 20
    \item prefsrOtherLep\_pt $>$ 20
    \item prefsrLep\_absEta $<$ 2.5
    \item prefsrOtherLep\_absEta $<$ 2.5
    \item abs(prefsrV\_mass - 91.) $<$ 15
  \end{itemize}
  \item Event-level selected region for swap: $bottom\_sel = (nBhad \geq 1)$. No requirements on the B hadrons themselves.
  \item B hadrons are built from final-state GenPart bottom-hadron ID logic, as before.
  \item Corrected sample construction: nominal unselected + massive selected.
  \item Slides below show both preselection LHE, b-tagged jet, and B hadron diagnostics.
\end{itemize}
\end{frame}
