# knowledge/

Durable, cross-study facts — the *"what's true"* reference layer for the analysis.
Distilled from finished studies and verified operating knowledge; not a narrative.

- Timeless facts go here; per-study narrative goes in `../studies/<slug>/LOGBOOK.md`.
- When a logbook Finding proves durable across studies, **promote** it into the
  right topic file here (in the same session — don't defer).
- Claude's private memory (`~/.claude/…`) only *points* at these files; when the
  two disagree, this repo wins. See `../AGENTS.md` → "Three durable stores".
